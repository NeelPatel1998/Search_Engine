The java web service application can be accessed by other programming languages such as .Net and PHP. Java web service application perform communication through WSDL (Web Services Description Language). There are two ways to write java web service application code: SOAP and RESTful. Java Web Services API There are two main API's defined by Java for developing web service applications since JavaEE 6. 1) JAX-WS: for SOAP web services. The are two ways to write JAX-WS application code: by RPC style and Document style. 2) JAX-RS: for RESTful web services. There are mainly 2 implementation currently in use for creating JAX-RS application: Jersey and RESTeasy. We will see full tutorials of JAX-WS and JAX-RS in next page.
