This tutorial covers the features of Entity Framework using Code First approach. It also explains the new features introduced in Entity Framework 6. Audience This tutorial is designed for those who want to learn how to start the development of Entity Framework in their application. Prerequisites You should have a basic knowledge of Visual Studio, C#, and MS SQL Server to make the most of this tutorial.
